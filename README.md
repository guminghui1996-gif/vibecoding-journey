# vibecoding-journey

代码小白的 AI 产品构建记录。

Vibecoding 本身并没有意义，真正有价值的是，在做项目的过程中，那些判断过程和思考。

## 这个网站记录什么

- 每天的内部目标、概念类比、实操动作、知识卡和可复用 Prompt。
- 从 VoiceMore 开始，把一个模糊想法拆成需求、代码、验证和发布的过程。
- 真实踩坑、判断依据和修正方式。

## 第一版边界

- Astro 静态网站。
- Markdown 内容。
- Vercel 部署。
- 不做登录、评论、数据库、后台和自动同步。

## 隐私边界

公开内容不得包含：

- API Key、Token、环境变量或任何密钥。
- 原始录音、口述全文、剪贴板内容。
- 私人窗口截图、聊天记录、邮件、账号信息。
- 未经同意的用户反馈或可识别个人信息。

## 本地运行

```powershell
npm install
npm run dev
```

本地预览地址通常是：

```text
http://localhost:4321/
```

## 构建检查

```powershell
npm run build
```
